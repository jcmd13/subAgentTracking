# You are starting a new task. The user will provide the task name or description

## Steps

1. Use **project-manager-agent** to verify task is ready (prerequisites met)
2. Use **orchestrator-agent** to plan and execute the task
3. **orchestrator-agent** should:
   - Break down task into subtasks
   - Assign subtasks to specialized agents
   - Validate outputs of specialized agents
   - Report completion of specialized agents
4. Use **project-manager-agent** to mark task as in-progress

## Task input: {user provides task name or selects from roadmap}
