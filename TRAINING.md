## üìñ Training Structure (6-8 hours)

### Part 1: Foundations (2 hours)
1. **Module 1**: Installation & Setup (30 min)
   - Virtual environments, installation, verification
2. **Module 2**: Basic Concepts (45 min)
   - Event-driven architecture, agents, workflows
3. **Module 3**: Event System Deep Dive (45 min)
   - Custom events, payloads, handlers

### Part 2: Monitoring (2 hours)
4. **Module 4**: Real-Time Monitoring Setup (90 min)
   - WebSocket server, dashboard, metrics aggregation
5. **Module 5**: Analytics & Insights (90 min)
   - Pattern detection, cost analysis, recommendations

### Part 3: Advanced Features (2 hours)
6. **Module 6**: Fleet Monitoring (60 min)
   - Workflow tracking, bottleneck detection
7. **Module 7**: Full Integration (90 min)
   - Production integration, best practices

### Part 4: Hands-On Project (2-3 hours)
- **Refactoring Project**: Apply everything learned to real code
